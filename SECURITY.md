# Security

## Reporting a vulnerability

If you believe you have found a security vulnerability, please report it responsibly:

- **Do not** open a public GitHub issue for security-sensitive bugs.
- Email a description of the issue and steps to reproduce (if possible) to the maintainers. You can reach us via the [GitHub repository](https://github.com/csvlinter/csvlinter) (e.g. through the "Security" tab → "Report a vulnerability", or by contacting the maintainers listed there).

We will acknowledge your report and work with you to understand and address the issue. We may ask for more detail and will keep you updated on the status of a fix. We appreciate your help in disclosing vulnerabilities in a responsible way.

## Supported versions

We release security fixes for the current major version. Upgrading to the latest release is recommended.
